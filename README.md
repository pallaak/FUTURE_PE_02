# FUTURE_PE_02 – AI Prompt Engineering Project

## Business
GlowClear Face Wash (D2C Skincare Brand)

## Objective
To generate AI-powered marketing content including UGC ads and website copy.

## Content Included
- UGC Ad Content (hooks, scripts, CTAs)
- Website Copy (homepage, services, CTA)

## Tools Used
ChatGPT (AI Content Generation Tool)

## Prompt Logic
Prompts were designed to:
- Define target audience (acne-prone users)
- Generate structured marketing content
- Ensure persuasive and conversion-focused writing
- Adapt tone for brand communication

## Output Summary
The final output includes UGC advertising content and complete website copy suitable for a D2C skincare brand.

## Note
This project demonstrates real-world application of prompt engineering for digital marketing content creation.
